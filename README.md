# Medical-AI-Assistant-Fine-Tuning-with-QLoRA-and-Unsloth
A medical AI assistant fine-tuning pipeline using QLoRA and Unsloth. Built with Python and HuggingFace TRL on Google Colab, it features 4-bit NF4 quantization, Low-Rank Adaptation (LoRA), and automated training visualization on the medalpaca/medical_meadow_medqa dataset.
